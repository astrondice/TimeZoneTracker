# 🚀 Overview

World Clock & Alarm Hub is a modern web application that combines:

- 🌍 Real-Time World Clocks
- ⏰ Smart Alarm System
- 📊 Alarm History Tracking
- ⚡ Express.js Backend
- 💾 Persistent Data Storage

Designed to help users manage multiple time zones and important reminders from a single dashboard.

# ⚡ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/world-clock-alarm-hub.git
```

## Install Dependencies

```bash
npm install
```

## Start Server

```bash
npm start
```

## Development Mode

```bash
npm run dev
```
