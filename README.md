# ⚡ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/world-clock-alarm-hub.git
```

## Install Dependencies

```bash
npm install
```

## Start Server

```bash
npm start
```

## Development Mode

```bash
npm run dev
```

# 🌐 API Endpoints

### Save Alarm Log

```http
POST /api/alarms/log
```

### Get Alarm Logs

```http
GET /api/alarms/logs
```
